# Call & Email Dashboard

This project is a React-based dashboard built using Context API, reusable components, Chart.js pie charts, and dynamic tables.  
It includes user selection, segment-based analytics, pagination, and clean UI matching assignment requirements.

## Features
- User dropdown with territory-based filtering
- Interactive pie chart with slice click behavior
- Right-side dynamic details table
- Permanent territory account details table
- Modular component-based architecture
- Fully responsive UI
